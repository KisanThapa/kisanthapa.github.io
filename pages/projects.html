Projects

Selected work focusing on leveraging computational techniques for biological insight.

CausalPath Webserver

Active

CausalPath analyzes molecular profiles (phosphoproteomics, transcriptomics, etc.) to infer causal biological mechanisms by integrating prior knowledge from pathway databases. It helps explain experimental observations in the context of known biology.

Tech: Java, JavaScript, HTML, CSS, Cytoscape.js
Live Webserver GitHub Project

Cell Activity Inference Landscape (KALE)

Ongoing

This project implements rank-based methods (like GSEA, VIPER) within a webserver to predict transcription factor (TF) activity from gene expression data. It provides biologists with an accessible tool to infer regulatory changes.

Tech: Python, Flask, Pandas, NumPy
Live Webserver GitHub Project
